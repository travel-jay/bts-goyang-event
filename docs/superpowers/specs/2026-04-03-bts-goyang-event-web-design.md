# Treasure Hunt for ARMY — 설계 문서

## 1. 개요

**서비스명: Treasure Hunt for ARMY**

BTS 콘서트 관람객(한국인 + 외국인)을 대상으로, 콘서트 티켓 제시 시 이벤트/혜택을 제공하는 매장 약 120곳의 리스트를 온라인 웹에서 쉽게 확인할 수 있는 반응형 웹사이트를 제작한다.

- **타겟**: 한국인 + 외국인 동등
- **운영 기간**: 콘서트 기간 한정 (일회성)
- **언어**: 한국어 + 영어

## 2. 기술 스택

- **구조**: 단일 `index.html` 파일 (HTML + CSS + JS 인라인)
- **지도**: Kakao Maps JavaScript SDK
- **지오코딩**: Kakao REST API (빌드 시 주소→좌표 변환, JSON에 미리 내장)
- **외부 지도 연결**: 네이버지도(한국어 모드) / Google Maps(영어 모드)
- **호스팅**: GitHub Pages (무료)
- **프레임워크**: 없음 (Vanilla JS)

## 3. 데이터 구조

### 3.1 원본 데이터
- 엑셀 파일: 한글/English 2개 시트
- 컬럼: 번호, 업체명, 업종, 주소, 이벤트 내용
- 약 120개 업체 (고양시 일산 일대)

### 3.2 카테고리 매핑
엑셀의 업종을 4개 카테고리로 재분류:

| 카테고리 | 포함 업종 |
|---|---|
| 음식 | 한식, 카페, 베이커리, 레스토랑, 치킨, 아시안, 식음료, 피자 등 |
| 쇼핑 | 대형마트, 복합쇼핑몰, 할인마트, 패션, 굿즈/액세서리, 한과 등 |
| 액티비티 | 성형외과, 포토부스 등 체험/서비스 업종 |
| 기타 | 위 3개에 해당하지 않는 업종 |

### 3.3 JSON 스키마
```json

  "nameKo": "롯데마트 화정점",
  "nameEn": "Lotte Mart Hwajeong Branch",

  "eventKo": "외국인 고양콘서트 티켓 제시 시 롯데상품권 5천원권 증정...",
  "eventEn": "Foreign visitors who present a Goyang concert ticket receive...",

```

## 4. UI 설계

### 4.1 페이지 구성
1. **헤더**: 제목 + KO/EN 언어 전환 토글
2. **이벤트 참여 안내**: 아코디언(접이식), 티켓 제시 방법 등 안내
3. **카테고리 필터**: [전체] [음식] [쇼핑] [액티비티] [기타] 탭
4. **지도 영역**: Kakao Map, 마커 + 클러스터링
5. **업체 카드 리스트**: 스크롤 탐색, 외부 지도 링크 포함

### 4.2 모바일 레이아웃 (기본)
- 세로 단일 컬럼
- 지도(50vh) → 카드 리스트 순서
- 필터 탭은 가로 스크롤
- 이벤트 안내는 기본 접혀있음

### 4.3 PC 레이아웃 (768px 이상)
- 좌측: 카드 리스트 (스크롤)
- 우측: 지도 (고정, sticky)
- 이벤트 안내는 기본 펼쳐진 상태

### 4.4 인터랙션
- **필터 탭 클릭** → 카드 리스트 + 지도 마커 동시 필터링
- **지도 마커 클릭** → 해당 업체 카드로 스크롤 + 하이라이트
- **카드 클릭** → 지도에서 해당 마커 중심으로 이동
- **외부 지도 버튼** → 한국어 모드: 네이버지도 / 영어 모드: Google Maps

### 4.5 마커 스타일
- 카테고리별 색상: 음식(주황), 쇼핑(보라), 액티비티(파랑), 기타(회색)
- 인접 마커는 클러스터링 (숫자로 묶어 표시)

## 5. 언어 전환

- KO/EN 토글 버튼 (페이지 리로드 없이 DOM 텍스트 교체)
- 브라우저 언어 감지로 초기 언어 자동 설정 (`navigator.language`)
- 외국인 브라우저 → 자동 영어 / 한국 브라우저 → 자동 한국어

## 6. 외부 지도 연결

- **한국어 모드**: 네이버지도 (`map.naver.com/v5/search/{주소}`)
- **영어 모드**: Google Maps (`google.com/maps/search/{주소}`)
- 모바일: 해당 지도 앱이 설치되어 있으면 앱으로 자동 연결

## 7. 배포

- GitHub Pages 무료 호스팅
- 레포에 `index.html` push → 자동 배포
- URL: `https://{username}.github.io/bts-goyang-event/`

## 8. 사전 준비 사항

- **Kakao Developers** 앱 등록 및 JavaScript 키 발급 (사용자 직접)
- 엑셀 데이터 → JSON 변환 (빌드 스크립트)
- Kakao REST API로 120개 주소 지오코딩 (빌드 시 1회)

## 9. 성공 기준

- PC/모바일 반응형으로 동작
- 한국어/영어 전환이 매끄럽게 작동
- 카테고리 필터가 지도 + 리스트에 동시 적용
- 지도 마커 ↔ 카드 리스트 연동
- 외부 지도 앱 연결 정상 동작
- GitHub Pages에서 정상 접속 가능
